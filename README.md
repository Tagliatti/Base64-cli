# Base64-cli
A tool for rapidly encode or decode base 64 in command line.

### Install
Install the module globally
```sh
npm install -g base64-cli
```

### Encode
```sh
base64cli -e test
```
or
```sh
base64cli --encode test
```

### Decode
```sh
base64cli -d dGVzdA==
```
or
```sh
base64cli --decode dGVzdA==
```
